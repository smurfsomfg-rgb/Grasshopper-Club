Las imágenes de las cartas de Tarot Rider-Waite se deben colocar en esta carpeta.

Puedes descargarlas desde:
https://github.com/andrewnester/tarot-api-js/tree/master/cards

Nombres de archivo recomendados: 
- 0_the_fool.jpg
- 1_the_magician.jpg
- ...
- king_of_wands.jpg

Asegúrate de que los nombres coincidan con los usados en el código JS para que se muestren correctamente.
